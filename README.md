# Meine Next.js Website

Eine einfache Next.js Website.

## Installation

Führen Sie zuerst folgendes aus, um alle benötigten Pakete zu installieren:

```bash
npm install
```

## Entwicklung

Um die Website im Entwicklungsmodus zu starten:

```bash
npm run dev
```

Öffnen Sie dann [http://localhost:3000](http://localhost:3000) in Ihrem Browser.

## Produktion

Um die Website für die Produktion zu erstellen:

```bash
npm run build
npm start
```
